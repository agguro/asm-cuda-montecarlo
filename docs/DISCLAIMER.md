# Model Disclaimer

This engine is a numerical simulation tool.

It models hypothetical price evolution under a Geometric Brownian Motion assumption using statistically derived parameters from historical data.

The simulation is not:

- a prediction engine
- financial advice
- a guarantee of future market behavior
- a representation of real market dynamics

Results are probabilistic estimates generated from simplified stochastic assumptions.

Real financial markets contain effects not modeled by this engine, including:

- structural regime changes
- liquidity shocks
- correlated market failures
- non-Gaussian events
- geopolitical events
- discontinuous price gaps
- behavioral effects

The engine should therefore be treated as an analytical instrument rather than an oracle.
